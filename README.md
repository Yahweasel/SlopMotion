If you just want to watch SlopMotion, go to https://twitch.tv/slopmotion
(assuming it's still up!)

This is SlopMotion. An AI slop generator and/or kinetic AI toy, intended for
streaming. It has an AI LLM generate code that produces visually interesting
images, and runs it on loop forever. It's intended to be used for streaming to,
e.g., Twitch.

It can also (arguably, mainly) take suggestions for AI slop to make from
viewers.

Copy the `.json.example` files to `.json` and configure them.

Run `server.mjs` to run the server, then connect a browser to
`http://localhost:17591`.
